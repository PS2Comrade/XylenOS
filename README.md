# XylenOS
![XylenOS logo](https://i.imgur.com/v5dNEFR.png) <br>
XylenOS (from Greek "ξυλοσ"), started as UselessOS or uOS, is a hobbyist project made by a Turkish and Bangladeshi Hobbyist. <br/>
